# 风格主题

同一个文档中的多张图表应保持视觉一致。主题定义了色彩系统的骨架——具体的可视化手法仍然自由发挥。

## 主题选择时机

- **首次生成图表时**：根据数据语境推荐一个主题，简短告知用户（如"使用 Ocean Blue 主题"），不必追问
- **后续同文档**：沿用上次主题。如果用户之前生成过图表，使用相同主题
- **用户可以随时切换**：说"换个风格"、"用暖色调"等，按描述匹配最近的主题

## 主题的使用方式

主题定义的是**色彩骨架**，不限制布局和可视化手法。具体来说：

**用主题统一的**：
- Header 渐变色
- 主色调条形图、环形图的填充色
- Badge/标签的底色
- 文字颜色层级
- 背景和分隔线

**仍然自由发挥的**：
- 布局结构（单栏/多栏/网格）
- 可视化手法选择（条形/环形/漏斗/热力格）
- 信息密度和组合方式
- 间距和字号
- 数据洞察和表达方式

## 根据数据语境推荐主题

| 数据类型 | 推荐主题 |

| 业务数据、KPI、通用分析 | Ocean Blue |
| 用户增长、运营、营销 | Warm Sunset |
| 技术架构、系统设计 | Slate Minimal |
| 教育、健康、ESG | Forest Green |

### Midnight Glass

暗色毛玻璃风格，适合高端数据展示、夜间场景、技术/科技感强的汇报。深色背景下的发光卡片让数据显得更精致。

```
主色:     #60a5fa (天蓝)
Header 渐变: linear-gradient(135deg, #60a5fa, #a78bfa, #f472b6) — 多色渐变标题文字
强调色:   #a78bfa (紫)
次强调:   #f472b6 (粉)
成功:     #4ade80
警告:     #fbbf24
危险:     #f87171
背景:     #0f1729
卡片:     linear-gradient(145deg, #1a2540, #162035) — 带 rgba(255,255,255,0.06) 边框
分隔线:   rgba(255,255,255,0.08)
文字层级: #f1f5f9 → #cbd5e1 → #94a3b8 → #64748b
Badge 底色: rgba(96,165,250,0.15) (蓝), rgba(74,222,128,0.15) (绿), rgba(251,191,36,0.15) (黄), rgba(248,113,113,0.15) (红)
多色系统: #60a5fa(蓝) #a78bfa(紫) #4ade80(绿) #fbbf24(黄) #f472b6(粉) #fb923c(橙) — 各色带同色 0.15 透明度的 box-shadow 发光
```

**暗色主题特殊技巧**：
- **毛玻璃卡片**：`background: linear-gradient(...)` + `border: 1px solid rgba(255,255,255,0.06)` + `box-shadow: 0 4px 24px rgba(0,0,0,0.3)`
- **渐变文字**：`background: linear-gradient(...); -webkit-background-clip: text; -webkit-text-fill-color: transparent`
- **发光效果**：数据色块加 `box-shadow: 0 0 12px rgba(色, 0.3)` 营造科技感
- **进度条/条形图**：底色用 `rgba(255,255,255,0.06)` 而非 `#f3f4f6`
- **表头**：`background: rgba(255,255,255,0.04)` 而非浅灰

## 根据数据语境推荐主题

| 数据类型 | 推荐主题 |
|---------|---------|
| 业务数据、KPI、通用分析 | Ocean Blue |
| 用户增长、运营、营销 | Warm Sunset |
